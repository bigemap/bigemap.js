L.bigemap.sanitize(string)

A HTML sanitization function, with the same effect as the default value of the sanitizer option of L.bigemap.featureLayer, L.bigemap.gridControl, and L.bigemap.legendControl.

Options Value Description
text string String of content you wish to sanitize.
