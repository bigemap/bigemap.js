L.bigemap.config.FORCE_HTTPS

By default, this is false. Bigemap.js auto-detects whether the page your map is embedded in is using HTTPS or SSL, and matches: if you use HTTPS on your site, it uses HTTPS resources.

Setting FORCE_HTTPS to true makes Bigemap.js always require HTTPS resources, regardless of the host page's scheme.

Example:

L.bigemap.config.FORCE_HTTPS = true;
